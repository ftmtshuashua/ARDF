ARDF是一个帮助开发者快速构建项目的基础框架,提供了丰富的可拓展的基础功能,简单配置即可开始业务开发.
GitHub：
https://github.com/ftmtshuashua/ARDF
交流和建议(QQ群)：759081198

----- 已提供的解决方案 -----
FragmentControl:控制Fragment的加载/显示/切换。优化性能预防内存泄漏
RadioGroupControl:加强版的RadioGroup，让任何布局带有RadioGroup的功能
